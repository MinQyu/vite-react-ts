# Custom Vite Template

React 학습 시작을 위한 사용자 정의 템플릿

```sh
degit MinQyu/vite-react-ts <폴더_이름>
cd <폴더_이름>
npm install
```

## 실행 명령어

개발 서버 구동

```sh
npm dev
```

프로젝트 빌드

```sh
npm build
```

프로젝트 프리뷰

```sh
npm preview
```

린팅 검사

```sh
npm lint
```

포멧팅

```sh
npm format
```

pre-push 린팅 & 포멧팅

```sh
npm lint-staged
```
